# Temple of Gains

Este é o repositório do site **Temple of Gains**, refatorado para uma estrutura limpa, moderna e performante.

## Estrutura de Ficheiros

- `index.html`: Página principal (Landing Page).
- `pages/`: Contém as páginas secundárias do site.
  - `sobre-nos.html`: Informações sobre a filosofia do templo.
  - `biblioteca.html`: Recursos e guias educativos.
- `css/`: Contém os estilos globais.
  - `style.css`: Design responsivo e moderno com tema escuro.
- `assets/`: Pasta para imagens e ícones.

## Melhorias Realizadas

1. **Nomes de Ficheiros:** Removidos espaços e caracteres especiais das URLs (ex: `biblioteca.html` em vez de `Temple of Gains - Biblioteca do Templo.html`).
2. **Código Limpo:** Removido todo o código desnecessário do Google Sites, resultando em ficheiros muito mais leves e fáceis de editar.
3. **CSS Externo:** Estilos centralizados num único ficheiro, facilitando mudanças de design em todo o site.
4. **Design Responsivo:** O site agora adapta-se corretamente a dispositivos móveis.
5. **SEO:** Estrutura HTML semântica para melhor indexação nos motores de busca.

## Como Publicar no GitHub Pages

1. Vá às **Settings** do seu repositório no GitHub.
2. No menu lateral, clique em **Pages**.
3. Em **Build and deployment**, selecione a branch `main`.
4. Clique em **Save**.
